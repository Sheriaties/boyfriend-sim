# 养成小男友 🏠💕

一个用 HTML + CSS + JS 制作的横版点按恋爱养成小游戏。像素风 + Love Choice 蜡笔风混搭，画面素材由 AI 生成。

## 玩法

- 顶部四条数值：**心动 / 激情 / 了解度 / 压力**
- 点场景内的物品（沙发、电脑、床、玩偶、水槽、淋浴、马桶……）跟男朋友互动
- 每次互动 = 3 小时游戏时间；到深夜男主会说想睡觉，自动跳次日早上
- 场景：客厅 ⇄ 卧室（夹层） ⇄ 厕所，右上外出触发主线
- 主线：外出 3 次触发夜色散步 CG（4 秒不选自动牵手），第 2 周固定触发争吵剧情
- 隐藏 Bad End：连续喂男主 10 次薯片 → 「肥肥的男孩子没人爱」

## 运行

用浏览器打开 `index.html` 即可。存档在 localStorage。

## 开发者模式

游戏内右上角有一个 ⚙ 齿轮按钮，或按 `~` 键，可以：
- 直接触发任一剧情
- 拖数值滑块
- 快进时间 / +1天 / 跳到第2周
- 外出/薯片次数 +1
- 显示热区调试框
- 重置存档

> 需要发布正式版时请把 `#dev-toggle`、`#dev-panel`、以及所有 `dev_*` 函数删掉。

## 素材说明

图片素材通过 OpenRouter API 调用 `google/gemini-3.1-flash-lite-image` 生成，白底后处理为透明。
